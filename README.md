# 🤖 Vision-Based Robotic Sorting System

A complete industrial automation project integrating **Machine Vision, PLC, and Robotics** to perform intelligent object sorting based on color and position.

This system demonstrates a real-world **Industry 4.0 smart sorting solution** using:

- ABB IRB 1200 Robot
- Mitsubishi FX5U PLC
- SICK InspectorP Vision Sensor
- Delta VFD Conveyor System

---

## 🔍 Overview

This project implements an automated pipeline where objects are:

1. Detected on a conveyor
2. Analyzed using a vision system
3. Processed via PLC logic
4. Picked & sorted by a robotic arm into different bins
